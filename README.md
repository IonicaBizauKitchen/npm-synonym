# npm-synonym

Turn your three-word-phrase into an abbreviation of n-p-m.

## Usage

```sh
$ npm install npm-synonym --global

$ npm-synonym strange strange strange
[ 'new',
  'non compos',
  'non compos mentis',
  'not all there',
  'not right',
  'novel',
  'nutty' ]
[ 'passing strange',

  'phenomenal',
  'prodigious',
  'psycho',
  'puzzling' ]
[ 'mad',
  'maddened',
  'maggoty',
  'manic',
  'marvelous',
  'mazed',
  'mental',
  'mentally deficient',
  'meshuggah',
  'miraculous',
  'moon-struck',
  'mysterious' ]

$ npm-synonym big system program
[ 'name', 'noble', 'noble-minded', 'nubile', 'numerous' ]
[ 'Ptolemaic universe',
  'pattern',
  'peace',
  'physique',

  'plenum',

  'procedure',
  'proceeding',

  'program of action',

  'pulsating universe' ]
[ 'Marshall Plan',

  'manifesto',
  'mapping',
  'master plan',
  'menu',
  'method',
  'methodize',

```
